# ForensicLens: Advanced Image Analysis Tool

ForensicLens is a professional digital forensics tool designed for investigators and cybersecurity students to verify image integrity and detect manipulation.

##  Features
* **Forensic Hashing:** MD5 & SHA-256 generation for Chain of Custody.
* **Metadata Extraction:** Full EXIF data analysis (GPS, Camera, Software).
* **Error Level Analysis (ELA):** Identifies digitally modified areas in an image.
* **Rich UI:** Professional terminal dashboard with tables and progress bars.
